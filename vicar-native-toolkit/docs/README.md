# VICAR Native Toolkit Documentation

Documentation for the **VICAR Native Toolkit** — native-like command execution
for VICAR tools running inside a long-lived Docker sidecar container.

New here? Start with the **[Walkthrough](VICAR_NATIVE_TOOLKIT_WALKTHROUGH.md)** or the
**[Quick Reference](QUICKREF.md)**.
